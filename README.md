# Chat App

A full functional real-time chat application built with Node.js, Express, and Socket.IO.

## Features

- Real-time messaging
- Multiple users support
- Online user list
- Typing indicators
- User join/leave notifications
- Responsive design

## Installation

1. Install dependencies:
```bash
npm install
```

2. Start the server:
```bash
npm start
```

3. Open your browser and go to `http://localhost:3000`

## Usage

1. Enter your username to join the chat
2. Start chatting with other users in real-time
3. See who's online in the sidebar
4. View typing indicators when others are typing

## Technologies Used

- Node.js
- Express.js
- Socket.IO
- HTML5
- CSS3
- JavaScript
